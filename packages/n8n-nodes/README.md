# n8n-nodes-nocioun

Google Contacts와 Notion을 위한 n8n 커스텀 노드 모음입니다.

## 설치

```bash
npm install n8n-nodes-nocioun
```

## 포함된 노드

### Google Contacts Node

- 연락처 목록 조회
- 연락처 검색
- 연락처 생성/수정/삭제

### Notion Node

- 데이터베이스 페이지 생성/수정/삭제
- 페이지 조회
- 데이터베이스 쿼리

## 인증 정보

### Google Contacts API

- OAuth 2.0 또는 Service Account 인증 지원

### Notion API

- Internal Integration Token 사용

## 사용 방법

1. n8n에서 노드를 설치한 후
2. 각 노드의 인증 정보를 설정
3. 워크플로우에서 노드를 사용
